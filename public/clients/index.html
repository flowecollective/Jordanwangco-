Privacy. This hub and every brief are set to noindex, so search engines skip them. They are still reachable by anyone with the URL. Give each client an unguessable filename (for example /clients/sb-7f3a91.html rather than a name), and consider a password on the /clients folder if your host supports it. Don't link this hub from your public site. To add a client: duplicate client-brief-template.html, rename it to the client's slug, fill it in, then copy a card above and point it at the new file.
